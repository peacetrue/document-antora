== 连接外网

刚装好的系统有个问题，无法连外网，这是由于 ifcfg 配置文件中的 ONBOOT 属性默认设置为 no。

* $ ip addr

image::ip-addr.png[]

$ vi /etc/sysconfig/network-scripts/ifcfg-enp0s3

[source%nowrap,shell]
----
ONBOOT=yes
----
